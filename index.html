考研日记

发表于 2019-03-15

写在前面

话说，一直在想要准备考研，然而一直在拖，自己都有点受不了自己的拖延症了。终于，在今天，下定决心开始准备考研了。为了不让自己懈怠，希望自己能够每天在这里写点东西，算是一天的总结了，同时也算是提醒自己。

3.15

今天算是准备考研的第一天（第一天开始看数学）。然而课内作业实在是太多了，数学也没看多少，但看了就算迈出第一步了。还是要抓紧写完作业。

我也不知道为啥选择今天开始准备考研，但既然开始了，那就坚持下去吧。

3.16

第二天：背单词，看数学，做作业。课内的作业真心感觉压力好大，虽然考研，但也没打算放弃课内的任务。不过，今天的复习时间确实很少。

有想过，先把复习放放，全力把作业啥的都整明白了，但才第二天，不想计划就这样断掉。

3.17

第三天：一天忙着补作业。。。下午去找King吉他们一起写作业，被热情款待，King吉做的饭真好吃。

不小心把人家厕所整堵了，事实证明，套保鲜膜并不好使。。。（深深的罪恶感。。。）

3.18

作业是越写越多，老师们是不会让你闲着的。

3.19

背单词、看数学、做作业、写代码。

3.20

上了一天课，写了一天作业。还好本周的作业马上写完了

3.21

3.22

3.23

3.24

3.25

2018 年终总结

发表于 2018-12-31

写在前面

今年，突然想认认真真地写一次年终总结了。不是很喜欢在 csdn 这种技术型的平台上写记录生活的博客，昨天利用 Hexo 搭好自己博客，想着今天写点东西。写的时候，才发现，自己这一年是真的没有做什么事情。虽然没干啥事，但 2018 年，对我来说，意义还是挺大的。

2018 上半年

上半年的状态说实在的，可以说不是很好。上半年主要做的事，除了吃喝玩乐了，就是睡觉了。偶尔高兴，写两行代码，很是佛系。但也不能说啥也没做，玩的时候，也还是做了点玩的事情。

  1. 上半年的主旋律就是看电影。特别喜欢看电影，从年初开始，看了《哈利波特》、《速度与激情》、《雷神》、《复联》……太多了，不列举了。反正就是刷了一堆电影，生活特别悠闲。

  2. 年初的时候，本来打算做个项目，学了一段时间的安卓。但学艺不精，项目无从下手，放弃了，挺打击自信心的。后来就没碰过 Android 开发。

  3. 老姐结婚了。借此机会，去了趟西安。第一次坐飞机，有点忐忑，一直担心会坠机，每次安全下飞机都感觉很庆幸。西安还是挺好玩的，感受到了浓浓的道教文化，但饮食还是有点不适应，几乎每顿饭都是粥和馒头，内心相当排斥了。不过那的面条还不错，可惜就吃了一顿。

  4. 上半年一共出了两次省，西安是一次，第二次是去大连。寝室五人组团去大连玩了几天，貌似是清明节去的，印象最深的就是，贼冷。不过并不耽误，该浪还是要浪的。老姚头安排得，可以说非常周到了。

  5. 狗哥把易点俱乐部交给我和谌爷了。说实在的，一年时间，我也啥也没干，白白浪费资源了。自己就是个扶不起来的阿斗，内心其实挺愧疚的，明年就收拾收拾走人。

总结一下上半年，就是十分悠闲、安于现状，浑浑噩噩地耗着时间。自我感觉特别良好，也不知道自己哪里来的自信。没有目标，也没有斗志，很迷茫。

2018 下半年

下半年倒是做了点事情。很多想做的事情都是在下半年才开始做的。

  1. 首先把科二和科三考完了。认识了一群当时一起苦逼练车的伙伴们，练车的时候一起唠嗑玩耍，倒是其乐融融，现在见到还特别亲切。科四的时间安排总是和我的课程撞上，一直在拖，争取明年考完吧。

  2. 用 github 的学生礼包，租了一个 VPS，在上面搭了自己的梯子。本来是没打算搞的，奈何魏老哥的到期了，只能自己动手了。付给魏老哥一年的租金，就用了一学期，不知道给不给退差价（此处应该有表情/kb）。

  3. 开始有意识地经营自己的博客和 github，虽然也没几个star，但总比之前啥也不做要好。

  4. 莫名地成为分班后的班长。这个有点无奈，本来就想默默度过剩下两年大学生活的，对这个挺排斥的。自己是没有一点成为 leader 的能力，俱乐部的事情都没搞好，当班长就太虚了。不过既然已经当了，安心当着就行了，反正上面有导员顶着，下面有同学帮着。

  5. 开始加入大创项目的开发团队。虽说大创项目都结题了，但我一点代码都没写，有点不太好。虽然啥也不会，但还是边学边写，现学现卖。这学期也总算写了一些后端的接口+测试，也算是出了点力了吧。

  6. 开始关注股票。可能是受各种贸易战文章的影响，对股票市场很好奇。啥也不懂，10 月份放了 1000 块进去，想试试股票。到现在已经赔了 60+ 了。虽然中间有接近回本的时候，但心理总是放不下已经沉没的资本，总是想等它赚回来，大概这就是传说中的套牢吧。我把投股票的事情跟家里人说了，虽然爸妈没怎么说话，但我知道他们肯定是排斥这个的，不过他们是一直支持我的任何决定。反倒是老姐，一直说我不学好。

  7. 加入了姚老师的实验室。然而去了也不知道要干啥。后来老师给了个题目，现在还没开始做，要推到明年了。

  8. 成为了预备党员。大一提交的入党申请书，前天成为了预备党员。我都以为被党组织抛弃了呢，但这学期入党步伐有点大。

  9. 搭建了自己的博客，算是抓住了 2018 的尾巴。之前，博客一直都是放在 csdn 上的。不过，这种记录生活的，感觉不太适合放在上面。其实一直都想搞一个自己的，但就是没做，这次算是给自己找了个搭建博客的理由吧。

  10. 下半年最重要的是认识了很多人，以前认识的也重新认识了一遍。其中，最让我惊叹的就是安老哥了，他让我知道了世上竟然有这么表面的兄弟。

下半年的总结，下半年有了一些未来的方向，虽然该玩还是玩，不过也大概知道自己的目标是啥，没有上半年那么迷茫。但也挺迷茫的，不知道自己到底想要什么。

总结

2018 年，做了很多傻逼的事情，但同时也认识到了自身很多不足的地方。其实自己身边有很多优秀的人，希望自己能多去学习。但也没必要跟其他人去作对比，对比总会有伤害。可能自己很努力去做的事，还不如他们随便搞一搞。所以，做好自己就好。

2018 年最大的遗憾就是没有完整地做好一件事，总是想到啥就做啥，做一点就不做了。希望 2019 年能真正做好一件事。

2019 年，注定会是不一样的一年。期待明年的年终总结。
